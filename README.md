# Helium: The .VDB file Reader in written in Rust

This is Helium, described in the header. I am writing this to replace the python I had in my pathtracer, [LeadV3](https://github.com/kakuking/leadV3) used for heterogeneous mediums. 

I could not find a file specification for OpenVDB files, so I am following [a guide to write OpenVDB files](https://jangafx.com/insights/vdb-a-deep-dive), and trying to reverse-engineer a reader instead of a writer.
